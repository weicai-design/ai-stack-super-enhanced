## 🚀 技术栈总览

### 后端技术
- **框架**: FastAPI + Uvicorn
- **数据验证**: Pydantic
- **文件处理**: pypdf, python-docx, openpyxl, python-pptx
- **AI能力**: 智能NLP、语义分析、依赖推理
- **压缩**: LZ4
- **并发**: async/await

### 前端技术
- **Cursor扩展**: TypeScript + VS Code Extension API
- **Web界面**: HTML5 + CSS3 + JavaScript
- **可视化**: D3.js力导向图
- **HTTP**: Fetch API, Axios
- **UI**: 原生（无框架依赖）

### 数据存储
- **当前**: 内存存储（快速原型）
- **可扩展**: PostgreSQL, Redis, MongoDB, ElasticSearch
