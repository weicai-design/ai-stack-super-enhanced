├─ 中间: 主工作区

│  └─ 统计卡片
│
└─ 右侧: 详情面板
   ├─ 文档详情
   ├─ AI专家建议
   └─ 实时监控

```

在聊天框中用中文自然语言:
├─ "上传这个PDF文档"
├─ "帮我找关于AI的资料"
├─ "分析一下知识库质量"
├─ "构建知识图谱"

AI会自动:
├─ 理解您的意图
├─ 调用相应功能
├─ 展示结果
└─ 给出建议
```
