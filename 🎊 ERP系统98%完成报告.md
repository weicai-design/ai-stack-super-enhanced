## 💡 新模块使用场景

```
1. 访问采购管理-供应商
2. 查看所有供应商列表

4. 新增或编辑供应商信息
5. 查看准时率和合格率
```

---

### 场景2: 物料库存管理

```
1. 访问物料管理(MRP)
2. 查看所有物料库存
3. 识别库存不足的物料（红色显示）
4. 点击"补货"发起补货申请
5. 查看库存总值和周转率
```

---

### 场景3: 生产计划

```
1. 访问生产计划管理
2. 查看所有生产计划
3. 查看生产进度（进度条）
4. 对"计划中"的订单点击"开始生产"
5. 跟踪生产完成情况
```

---

### 场景4: 质量检验

```
1. 访问质量检验管理
2. 查看检验记录
3. 查看合格率统计
4. 新增检验记录
5. 分析质量趋势
```

---

### 场景5: 仓储管理

