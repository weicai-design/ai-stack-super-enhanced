## 🎊 使用流程

### 对于普通用户（推荐）

```
第1步: 访问主控制台
→ http://localhost:8000/

第2步: 选择功能
├─ 点击"API文档"查看所有功能
├─ 点击"RAG仪表板"查看RAG状态
├─ 点击"知识图谱"可视化图谱
└─ 点击"智能搜索"进行搜索

第3步: 在Swagger UI中操作
→ 使用"Try it out"测试功能
→ 填写参数，点击Execute
→ 查看返回结果

完全可视化操作，无需编程！
```

### 对于开发者

```
直接调用API:
curl http://localhost:8000/stock/data/000001
curl http://localhost:8000/finance/dashboard

或使用Python:
import requests
response = requests.get("http://localhost:8000/stock/data/000001")
```
