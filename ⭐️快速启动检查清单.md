## 🚀 10分钟启动服务

### 前置检查

```bash
# 检查Docker
docker --version
docker-compose --version

# 检查Python
python3 --version

# 检查端口
lsof -i :8011  # RAG
lsof -i :8013  # ERP
lsof -i :9090  # Prometheus
lsof -i :3000  # Grafana
```

### 快速部署（Docker方式）

```bash
# 1. 配置环境变量（1分钟）
cp .env.example .env
# 编辑 .env，填入必要配置

# 2. 创建网络（10秒）
docker network create ai-stack-network

# 3. 启动服务（5分钟）
docker-compose up -d

# 4. 检查状态（30秒）
docker-compose ps

# 5. 启动监控（2分钟）
cd monitoring
docker-compose -f docker-compose.monitoring.yml up -d

# 6. 验证服务（1分钟）
curl http://localhost:8011/health  # RAG
curl http://localhost:8013/health  # ERP
curl http://localhost:9090/-/healthy  # Prometheus
```
