## 🎯 快速开始指南

### 步骤1：环境准备

```bash
# 1. 克隆或下载项目
cd /Users/ywc/ai-stack-super-enhanced

# 2. 安装依赖（方式1：pip）
pip install -r requirements.txt

# 或使用Poetry（方式2：现代化）✨
poetry install

# 3. 配置环境变量
cp .env.example .env
# 编辑.env文件，设置必要的配置
```

### 步骤2：启动系统

```bash
# 方式1：直接启动
cd "📚 Enhanced RAG & Knowledge Graph"
source ../venv/bin/activate
python -m uvicorn api.app:app --host 0.0.0.0 --port 8000 --reload

# 方式2：使用启动脚本
./scripts/start_all_services.sh

# 方式3：使用Docker
docker-compose up -d

# 方式4：使用K8s ✨
kubectl apply -f infra/k8s/
```

### 步骤3：访问系统

```
🏠 主控制台:     http://localhost:8000/
📖 API文档:      http://localhost:8000/docs
✨ 增强功能:     http://localhost:8000/enhancements/health
📚 中文指南:     http://localhost:8000/guide
🏥 健康检查:     http://localhost:8000/health
```

### 步骤4：运行测试

```bash
# 运行所有测试
pytest tests/ -v

# 运行特定模块测试
pytest tests/finance/ -v
```

```bash

  -d '{

    "content": "这是一段需要记忆的对话内容",
```

```bash
    "content": "明天下午3点开会",
    "remind_time": "2025-11-10T15:00:00"
```
