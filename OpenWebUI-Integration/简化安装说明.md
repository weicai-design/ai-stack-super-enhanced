## 🎯 3步安装Functions

### Step 1: 进入Functions页面

**方式1**: 直接访问
```
http://localhost:3000/workspace/functions
```

**方式2**: 通过菜单
1. OpenWebUI主页
2. 点击左下角 👤 用户头像
3. Settings → Functions

**首次访问需要登录**:
- 点击 "Sign up" 注册
- 填写任意信息即可（本地账号）

---

### Step 2: 上传第一个Function

在Functions页面：

1. **点击 '+' 按钮**
2. **粘贴代码** (Command+V) ← RAG Integration已在剪贴板
3. **点击 'Save'**
4. **配置** (点击⚙️):
   ```
   rag_api_endpoint: http://host.docker.internal:8011
   ```
5. **启用** (开关变绿)

---

### Step 3: 重复6次

告诉我"继续"，我会复制下一个Function代码到剪贴板。

重复上面的粘贴、保存、配置步骤。
