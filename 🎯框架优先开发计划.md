## 🎯 框架内容

每个模块的框架包含：

### 1. 目录结构
```
module_name/
├─ __init__.py           # 模块初始化
├─ models.py             # 数据模型定义
├─ manager.py            # 核心管理器
├─ api.py                # API端点
└─ README.md             # 模块说明
```

### 2. 数据模型
- 核心实体定义
- 枚举类型
- 关系定义
- 验证规则

### 3. 管理器框架
- CRUD基础操作
- 核心业务逻辑接口
- 数据聚合方法
- 统计查询方法

### 4. API接口
- 健康检查
- 数据操作端点
- 查询分析端点
- 管理功能端点

### 5. 模块互联
- 与RAG集成接口
- 与交互中心集成
- 与租户系统集成
- 数据共享机制
