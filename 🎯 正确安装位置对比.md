# 🎯 OpenWebUI安装Plugin - 正确位置对比

## ❌ 错误位置（你当前所在）

```
设置 (Settings)
  └── 外部工具 (External Tools / Tool Servers)
      └── 管理工具服务器
          └── "添加连接" 弹窗
              ├── URL 输入框
              └── API密钥 输入框
```

**这是什么？**
- 用于连接外部API服务器
- 需要输入URL和API Key
- **不能粘贴Python代码**

## ✅ 正确位置（应该去这里）

```
设置 (Settings)
  └── 工作空间 (Workspace)
      └── Functions  ← 找这个！
          └── 点击 "+" 按钮
              └── 代码编辑器（大框框）
                  └── 在这里粘贴Python代码
```

**这是什么？**
- OpenWebUI的Functions功能
- 可以直接粘贴Python代码
- **这才是安装Plugin的地方**

**为什么正确？**
- 有完整的代码编辑器
- 支持Python Plugin

## 🔧 正确的安装步骤

### 第1步：进入Functions页面

访问：http://localhost:3000/workspace/functions

### 第2步：点击 "+" 按钮

在Functions页面右上角，有一个 **"+"** 按钮，点击它

### 第3步：粘贴代码

出现代码编辑器后：
1. 点击编辑区域
2. 按 **Command+V** 粘贴
3. 会看到Python代码（以 `"""` 开头）

### 第4步：保存

点击底部的 **"Save"** 按钮

### 第5步：配置

在Functions列表中找到新创建的Function，点击 ⚙️ 配置
