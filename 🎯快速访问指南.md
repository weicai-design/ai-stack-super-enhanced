# 🎯 快速访问指南

## 🚀 两种访问方式

### 方式1: 双击HTML文件（推荐）⭐

在Finder中找到这两个文件，双击即可：

```
📖API中文操作指南.html  →  打开详细的中文使用教程
🚀打开API文档.html      →  打开API交互式文档
```

### 方式2: 直接访问网址

在浏览器中访问：

```
中文操作指南: http://localhost:8000/guide
API文档:     http://localhost:8000/docs
健康检查:     http://localhost:8000/health
```

## 📖 推荐使用流程

### 第一次使用

1. **先打开** `📖API中文操作指南.html`
2. **阅读**"界面术语对照表"（2分钟）
3. **阅读**"快速开始"章节（5分钟）
4. **然后打开** `🚀打开API文档.html`
5. **对照指南**开始测试API

### 日常使用

1. **双屏模式**（推荐）
   - 左屏：中文指南（参考）
   - 右屏：API文档（操作）

2. **单屏模式**
   - 先看指南了解步骤
   - 再打开文档实际操作
