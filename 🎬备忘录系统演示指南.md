## 📊 API测试（可选）

### 健康检查
```bash
curl http://localhost:8000/health
```

### 查看所有备忘录
```bash
curl "http://localhost:8000/api/v1/memos/?user_id=default_user"
```

### 手动创建备忘录
```bash
curl -X POST "http://localhost:8000/api/v1/memos/" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "测试备忘录",
    "type": "task",
    "priority": "high",
    "tags": ["测试"],
    "user_id": "default_user"
  }'
```

### 搜索备忘录
```bash
curl "http://localhost:8000/api/v1/memos/search/query?query=ERP&user_id=default_user"
```

### 查看统计信息
```bash
curl "http://localhost:8000/api/v1/memos/statistics/summary?user_id=default_user"
```

## 🔍 观察要点

### 1. 智能识别
- ✅ 关键词匹配（需要、完成、开发）
- ✅ 时间识别（明天、3点、下周）
- ✅ 优先级判断（自动或根据关键词）
- ✅ 标签提取（ERP、RAG、测试等）

### 2. 置信度机制
- **高置信度 (≥0.8)**: 🟢 自动创建，显示成功通知
- **中置信度 (0.5-0.8)**: 🟡 请求确认
- **低置信度 (<0.5)**: ⚪ 不处理

### 3. 通知行为
- 成功通知：5秒后自动消失
- 确认通知：10秒后消失或手动操作
- 错误通知：3秒后消失

### 4. 动画效果
- 通知滑入/滑出
- 侧边栏滑动
- 备忘录高亮
- 过期脉冲

## 🐛 常见问题

### Q1: 通知没有显示？
**解决**：
- 检查浏览器控制台（F12）
- 确认API服务运行正常
- 刷新页面重试

### Q2: 侧边栏不显示？
**解决**：
- 检查是否点击了浮动按钮📝
- 刷新页面
- 检查浏览器控制台错误

### Q3: 识别不准确？
**说明**：
- 当前是基于规则的识别
- 某些情况置信度较低会请求确认
- 可以调整识别阈值

### Q4: API连接失败？
**解决**：
```bash
# 检查API是否运行
curl http://localhost:8000/health

# 如果没有运行，重新启动
bash scripts/start_memo_demo.sh
```
