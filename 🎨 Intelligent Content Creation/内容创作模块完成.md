## 🚀 如何使用

### 启动API服务
```bash
cd "/Users/ywc/ai-stack-super-enhanced/🎨 Intelligent Content Creation"
pip install fastapi uvicorn beautifulsoup4 requests
python api/main.py
```

### 使用示例

**获取热点话题**:
```bash
curl http://localhost:8016/api/content/materials/hot-topics
```

**生成内容**:
```bash
curl -X POST http://localhost:8016/api/content/generate \
  -H "Content-Type: application/json" \
  -d '{"topic": "秋季穿搭", "platform": "xiaohongshu"}'
```

**创建内容计划**:
```bash
curl -X POST http://localhost:8016/api/content/plan/create \
  -H "Content-Type: application/json" \
  -d '{"topic": "美妆分享", "platforms": ["xiaohongshu", "douyin"]}'
```
