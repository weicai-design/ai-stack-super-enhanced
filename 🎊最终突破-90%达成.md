## 💡 技术要点

### Python 3.9+ 类型注解改进

在Python 3.9及以上版本，可以直接使用内置类型：

```python
# 旧方式 (需要导入)
from typing import List, Dict, Tuple
def foo(items: List[str]) -> Dict[str, int]: ...

# 新方式 (内置支持)
def foo(items: list[str]) -> dict[str, int]: ...
```

**优势**:
- 更简洁
- 无需导入typing
- 运行时开销更小
- Python官方推荐
