## 🔌 API接口

### 专家列表
- `GET /api/expert/` - 获取所有53个专家列表
- `GET /api/expert/count` - 获取专家统计信息

### 专家分析
- `POST /api/expert/analyze` - 使用指定专家进行分析

### 专家协同
- `POST /api/expert/collaboration/session` - 创建协同会话
- `POST /api/expert/collaboration/contribution` - 添加专家贡献
- `GET /api/expert/collaboration/session/{session_id}` - 获取会话详情
- `GET /api/expert/collaboration/sessions/active` - 获取活跃会话
- `GET /api/expert/collaboration/summary` - 获取协同统计摘要
- `POST /api/expert/collaboration/session/{session_id}/finalize` - 完成协同会话
