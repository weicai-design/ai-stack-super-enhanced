## 💡 使用指南

### 快速开始

#### 方式1: Docker部署（推荐）
```bash
# 1. 配置环境变量
cp .env.example .env
# 编辑.env填入API密钥

# 2. 一键启动
docker-compose up -d

# 3. 查看状态
docker-compose ps

# 4. 访问服务
open http://localhost
```

#### 方式2: 本地部署
```bash
# 1. 安装依赖
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 2. 启动服务
./scripts/quick_start.sh

# 3. 打开控制台
open unified-dashboard/index.html
```

### 服务访问

| 服务 | 地址 | 说明 |
|------|------|------|
| 统一控制台 | `file:///.../unified-dashboard/index.html` | 主控制台 |
| 数据可视化 | `file:///.../unified-dashboard/visualizations.html` | 实时图表 |
| AI交互中心 | `http://localhost:8020` | 聊天界面 |
| RAG系统 | `http://localhost:8011/docs` | API文档 |
| ERP系统 | `http://localhost:8013/docs` | API文档 |

### 常用命令

```bash
# 健康检查
python3 scripts/health_check.py

# 集成测试
python3 scripts/integration_test.py

# 查看日志
```
