## 🔧 常见问题

### Q1: 服务启动失败？

**解决方案**:
```bash
# 检查端口占用
lsof -i :8013
lsof -i :8000

# 杀死占用进程
kill -9 <PID>

# 重新启动
./start_erp.sh
```

### Q2: API调用失败？

**检查**:
1. 服务是否正在运行
2. 端口是否正确（8013）
3. 请求格式是否正确

**测试连接**:
```bash
curl http://localhost:8013/health
```

### Q3: 如何配置API密钥？

```bash
cd "🔧 API配置向导"
python3 api_key_configurator.py --guide
```

### Q4: 如何查看所有可用功能？

**访问**:
```bash
curl http://localhost:8013/api/advanced/status
```

或查看: http://localhost:8013/api/advanced/summary
