## ❓ 如果打不开

### 解决方案1: 检查服务

```bash
# 检查服务是否运行
ps aux | grep uvicorn

# 如果没有运行，启动服务:
cd "/Users/ywc/ai-stack-super-enhanced/📚 Enhanced RAG & Knowledge Graph"
source ../venv/bin/activate
python3 -m uvicorn api.app:app --host 0.0.0.0 --port 8000 --reload
```

### 解决方案2: 清除浏览器缓存

```
Chrome: Cmd + Shift + R (强制刷新)
Safari: Cmd + Option + R
Firefox: Cmd + Shift + R
```

### 解决方案3: 使用其他浏览器

```
如果Chrome打不开，试试:
- Safari
- Firefox
- Edge
```

### 解决方案4: 检查端口占用

```bash
# 检查8000端口是否被占用
lsof -i :8000

# 如果被占用，可以杀掉进程
kill -9 <PID>

# 然后重新启动服务
```

## 🔍 检查清单

如果Agent界面打不开，按照这个清单检查：

- [ ] 服务是否在运行？（ps aux | grep uvicorn）
- [ ] 8000端口是否可访问？（curl http://localhost:8000/health）
- [ ] Agent API是否正常？（curl http://localhost:8000/agent/experts）
- [ ] agent_chat.html文件是否存在？
- [ ] 浏览器是否能访问localhost？
- [ ] 防火墙是否阻止了访问？
- [ ] 是否需要清除浏览器缓存？
