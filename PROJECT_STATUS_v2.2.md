### 启动所有服务（一行命令）

```bash
# 启动核心服务 + 缓存 + 异步任务 + 搜索 + 监控
docker-compose up -d && \
docker-compose -f docker-compose.cache.yml up -d && \
docker-compose -f docker-compose.celery.yml up -d && \

docker-compose -f monitoring/docker-compose.monitoring.yml up -d

echo "✅ 所有服务已启动！"
```
